# 旅のしおり

旅行の行程表（しおり）を作成・共有できるWebアプリです。ビルド不要の静的ファイル構成で、ブラウザだけで動きます。

**公開URL:** https://krgok.github.io/tabi-no-shiori/

## 使い方

1. 行きたい場所をカテゴリー（移動🚌・食事🍽️・宿泊🏨・観光📸）ごとに追加し、ドラッグで並べ替え
2. 「移動手段の選択とルートを検討」を押すと各区間の移動手段と概算時間が自動入力されます（地図・行程番号ピン付き）
3. 「共有」ボタンのリンクを送るだけでメンバーと共有できます（日・英・中・タイの4言語対応）

## メモ

- しおりのデータはお使いのブラウザ内（localStorage）にのみ保存されます
- Google Maps APIキー（任意設定）も各自のブラウザにのみ保存され、共有リンクには含まれません
- 詳細仕様は [SPEC.md](SPEC.md) を参照
