# 旅のしおり

旅行の行程表（しおり）を作成・共有できるWebアプリです。ビルド不要の静的ファイル構成で、ブラウザだけで動きます。

**公開URL:** https://krgok.github.io/tabi-no-shiori/

## 使い方

1. 行きたい場所をカテゴリー（移動🚌・食事🍽️・宿泊🏨・観光📸）ごとに追加し、ドラッグで並べ替え
2. 「移動手段の選択とルートを検討」を押すと各区間の移動手段と概算時間が自動入力されます（地図・行程番号ピン付き）
3. 「共有」ボタンのリンクを送るだけでメンバーと共有できます（日・英・中・タイの4言語対応）

## 主な機能

- 行程の時刻自動計算、ドラッグ並べ替え、複製、行程番号と連動した地図ピン
- 持ち物リスト・やることリスト（ヘッダーの🧳からいつでも開ける）
- 🔒 非公開マーク（項目・メモ・リスト単位）— 共有時に除外される
- 時差対応（日ごとのタイムゾーン、移動での到着地タイムゾーン）
- PDF出力（🖨）、CSV入出力、複数しおり管理・アーカイブ
- Googleログイン（任意）でクラウド保存・端末間同期・公開URLの発行

## データの保存先とプライバシー

- **未ログイン時**: しおりはお使いのブラウザ内（localStorage）にのみ保存されます
- **ログイン時**: ご自身のGoogleアカウントに紐づけてクラウド（Firestore）に保存され、本人だけが読み書きできます
- **共有リンク・公開URL**: 🔒 非公開マークを付けた項目・メモは**除外されたコピー**が共有されます（データ自体が含まれません）
- **Google Maps APIキー**（任意設定）: 各自のブラウザ内にのみ保存され、共有リンク・CSVには含まれません

## 開発

ローカルで動かすには `サーバー起動.bat` をダブルクリック（または `python -m http.server 8934`）して http://localhost:8934 を開きます。
